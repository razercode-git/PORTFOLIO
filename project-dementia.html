Dementia Detection Tool

A web-based AI prototype to detect early-stage dementia from short cognitive tests and questionnaire responses. Built during the Smart India Hackathon (SIH) as a proof-of-concept to help clinicians with early screening.

Overview

This project collects simple cognitive test inputs (memory tasks, pattern recognition, timed tasks) and demographic/contextual questions, then runs a lightweight model inference to provide a risk-score and suggested next steps. The UI focuses on accessibility, clear instructions, and privacy — no personally-identifiable data is stored on the public demo.

Model & Technical Details

  • Model: Lightweight neural network trained on selected cognitive features (proof-of-concept; not a clinical diagnostic).
  • Backend: Flask app serving a TensorFlow model for inference.
  • Frontend: HTML/CSS/JavaScript for tests, results UI and accessibility features.
  • Privacy & Ethics: Designed as a screening aid only; users are encouraged to consult clinicians for interpretation.

SIH Entry

Presented as an SIH prototype focusing on early detection and community screening. The submission included a demo, a short validation plan, and notes on responsible deployment.
